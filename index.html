Department of Lost Circuits

A Generative Postage Series for Obsolete Electronics

Controls

  • S Save current sheet as PNG
  • R Reseed (new random border assignments)
  • ← → Navigate sheets
  • L Toggle RISO preview (BLACK+TEAL)
  • E Export RISO plates (when in RISO mode)

Each stamp is generated from device data including release year, category, and image.

Year → circuit density (older = denser)

Category → badge (audio/storage/gaming/computing/camera/phone)

Border varies per stamp (perforated / scalloped / zigzag / ticket)
